<h2>Descripción</h2>
Aplicación que permite consultar en wikipedia información de series y almacenarlas localmente utilizando SQLite.

Proyecto desarrollado bajo el marco de la materia Diseño y Desarrollo de Software, en el cual partiendo de una implementación base dada, se debía realizar las tareas del sprint actual:

- Rediseñar la arquitectura de la aplicación utilizando el patrón de diseño MVP
- Refactorizar todo el código siguiendo los principios SOLID y clean code.
- Implementar una historia de usuario sobre la posibilidad de puntuar series, y otras funcionalidades opcionales.
